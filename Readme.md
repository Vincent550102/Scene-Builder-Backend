# Scene Builder 

### Steps
1. Create `.env` file and configure as:
   
    ```bash
    USERNAME=[你要設定的 mongodb 帳號，沒有中括號]
    PASSWORD=[你要設定的 mongodb 密碼，沒有中括號]
    ```
2. `sudo docker compose up -d`
